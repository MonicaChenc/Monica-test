# Installation Guide

This guide explains a clean setup flow for contributors.

## 1. Clone the repository

```bash
git clone https://github.com/MonicaChenc/Monica-test.git
cd Monica-test
```

## 2. Install dependencies

```bash
npm ci
```

## 3. Run the docs site

```bash
npm run start
```

## 4. Verify production build

```bash
npm run build
```

If the build command passes, your environment is configured correctly.
