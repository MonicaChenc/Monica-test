# Getting Started

Use this page as a quick checklist to run the docs site locally.

## Prerequisites

- Node.js 20 or later
- npm 10 or later

## Install dependencies

```bash
npm ci
```

## Start local development

```bash
npm run start
```

The site will be available at `http://localhost:3000`.

## Build for production

```bash
npm run build
```

If build succeeds, your content is ready to deploy.
